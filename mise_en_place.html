mise en place du css

le css :, choisir la police , la taille la couleur des textes, choisir aussi la mise en page et tout

Où écrit-on le CSS ?

Dans un fichier .css (recommandé)

on ajoutes dans l'entête :

< link rel="stylesheet" href="nomDuFichier.css" />

Dans l'en-tête < head> du fichier HTML

insérer le code CSS directement dans une balise < style> à l'intérieur de l'en-tête < head>

Directement dans les balises (non recommandé)

vous pouvez ajouter un attribut style à n'importe quelle balise. Vous insérerez votre code CSS directement dans cet attribut

< p style="color: green;">Bonjour et bienvenue sur mon site !< /p>

Appliquer un style : sélectionner une balise

p{color: blue;}

balise{propriété: valeur;}

3 éléments :

  • des noms de balises : on écrit les noms des balises dont on veut modifier l'apparence. Par exemple, si je veux modifier l'apparence de tous les paragraphes < p> , je dois écrire p ;
  • des propriétés CSS : Il y a par exemple la propriété color qui permet d'indiquer la couleur du texte, font-size qui permet d'indiquer la taille du texte, etc.
  • les valeurs : pour chaque propriété CSS, on doit indiquer une valeur. Par exemple, pour la propriété color , il faut indiquer le nom de la couleur. Pour font-size , il faut indiquer quelle taille on veut, etc.

on peut appliquer un style à plusieurs balises

p,h1{color: blue;}

Appliquer un style : class et id

  • l'attribut class
  • l'attribut id
dans l'html

attribut que l'on peut mettre sur n'importe quelle balise, aussi bien titre que paragraphe, image, etc.

< h1 class="nomQonVeu1"> < /h1> < p class="nomQonVeu2"> < /p> < img class="nomQonVeu3" />

dans le css

.nomQonVeut1/2/3{color: blue;}

  • class : precède d'un .
  • id : precède d'un #

    appliquer un style : les sélecteurs avancés

    Les sélecteurs que vous connaissez déjà

    Je veux toucher tous les paragraphes

    p{}

    Tous les titres et tous les textes importants

    h1, em{}

    sélectionner des balises précises

    .class{}
    #id{}

    Les sélecteurs avancés

    * : sélecteur universel

    Sélectionne toutes les balises sans exception.

    *{}

    a b: une balise contenue dans une autre

    < em> situées à l'intérieur d'une balise < h3>

    h3 em{}

    a + b : une balise qui en suit une autre

    Sélectionne la première balise < p> située après un titre < h3>

    h3 + p{}

    a[attribut] : une balise qui possède un attribut

    Sélectionne tous les liens < a> qui possèdent un attribut title

    a[title]{}

    a[attribut="Valeur"] : une balise, un attribut et une valeur exacte

    idem, mais l'attribut doit en plus avoir exactement pour valeur « Cliquez ici »

    a[title="Cliquez ici"]{}

    a[attribut*="Valeur"] : une balise, un attribut et une valeur

    Idem, l'attribut doit cette fois contenir dans sa valeur le mot « ici » (peu importe sa position).

    a[title*="ici"]{}

    D'autres sélecteurs existent !
