Insérer des images

comment insérer l'image :

pour insérer une image il faut utiliser les balises "< img />"

deux attributs obligatoires :
src : mettre le chemin de l'image ou le lien
alt

L'attribut permettant d'afficher une bulle d'aide est le même que pour les liens : il s'agit de title . Cet attribut est facultatif. (passez la souris sur la polo3)

miniature cliquable

De nombreux sites permettent de redimensionner des images, comme ResizePixel , par exemple
Je les place toutes les deux dans un dossier. J'affiche la version mini sur ma page et je fais un lien vers la version normale , pour que l'image agrandie s'affiche lorsqu'on clique sur la miniature. cliquez sur la polo4
