Faites votre mise en page avec Flexbox

Un conteneur, des éléments

< div id="conteneur">
< div class="element 1">Element
< div class="element 2">Element
< div class="element 3">Element
< /div>

Soyez flex !

#conteneur {display: flex;}

La direction

le sens que l'on veut. Avec flex-direction

row :sur une ligne (par défaut) ;
column :sur une colonne ;
row-reverse : une ligne,ordre inversé ;
column-reverse :une colonne,ordre inversé.

#conteneur
{display: flex;
flex-direction: column;}

Le retour à la ligne

flex-wrap :

nowrap : pas de retour (par défaut) ;
wrap : plus la place , retour;
wrap-reverse :à la ligne en sens inverse.

Alignez-les !

axe principale : horizontale => axe secondaire : verticale
axe principale : verticale => axe secondaire : horizontale

Aligner sur l'axe principal

changer leur alignement
justify-content

flex-start : alignés au début (par défaut) ; flex-end : alignés à la fin ;
center : alignés au centre ;
space-between :étirés sur tout l'axe (de l'espace entre) ;
space-around : idem, mais aussi de l'espace sur les extrémités.

marche aussi dans une direction verticale

Aligner sur l'axe secondaire

align-items :

stretch :sur tout l'axe (valeur par défaut) ;
flex-start :au début ;
flex-end :à la fin ;
center :au centre ;
baseline :sur la ligne de base (semblable à flex-start).

#conteneur
{display: flex;
justify-content: center;
align-items: center;}

Aligner un seul élément

align-self :

#conteneur{
display: flex;
flex-direction: row;
justify-content: center;
align-items: center;

.element:nth-child(2) /* On prend le deuxième bloc élément */{
background-color: blue;
align-self: flex-end; /* Seul ce bloc sera aligné à la fin */ }

Répartir plusieurs lignes

un cas de figure où nous avons plusieurs lignes

align-content :

flex-start : placés au début ; flex-end : placés à la fin ; center :placés au centre ; space-between :séparés avec de l'espace ; space-around : idem, de l'espace au début et à la fin ; stretch (par défaut) : occuper tout l'espace.

Rappel à l'ordre

#conteneur {display: flex;}

.element:nth-child(1){order: 3;}
.element:nth-child(2){order: 1;}
.element:nth-child(3) {order: 2;}

Encore plus flex : faire grossir ou maigrir les éléments

.element:nth-child(2) {flex: 1;}

.element:nth-child(1){flex: 2;}
.element:nth-child(2){flex: 1;}
