Découvrez le modèle des boîtes

Les balises de type block et inline

block :

exemple des paragraphes
crée automatiquement un retour à la ligne avant et après
un site est constituée d'une série de blocs
il est possible de mettre un bloc à l'intérieur d'un autre

inline :

exemple des liens
à l'intérieur d'une balise block
pas de retour à la ligne

Les balises universelles

< span>< /span> (inline) ;
< div>< /div> (block).

Les dimensions

width :la largeur du bloc.
en (px), ou (%).
height :la hauteur du bloc.
en (px), ou (%).

Minimum et maximum

min-width : largeur minimale ;
min-height : hauteur minimale ;
max-width : largeur maximale ;
max-height : hauteur maximale.

p
{width: 50%;
min-width: 400px;}

Les marges

padding :marge intérieure. en (px) ;
margin : marge extérieure. Là encore, (px)

Centrer des blocs

marges extérieures automatiques : margin: auto;

Quand ça dépasse…

overflow : couper un bloc

visible : (par défaut) le texte reste visible ;
hidden : le texte sera coupé.
scroll : là encore, le texte sera coupé . Sauf que cette fois, barres de défilement
auto : le navigateur qui décide

word-wrap : couper les textes trop larges

p{word-wrap: break-word;}
