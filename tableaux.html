Ajoutez des tableaux

Un tableau simple

< table> indique le début et la fin du tableau

< tr> : indique le début et la fin d'une ligne du tableau ;
< td> < /td> : indique le début et la fin du contenu d'une cellule.

on ajoutes des bordures afin qu'elles soient visibles

td {border: 1px solid black; }

coller les bordures

border-collapse:

collapse : bordures collées
separate : bordures dissociées

table {border-collapse: collapse;}
td{border: 1px solid black;}

La ligne d'en-tête

< th>au lieu < td> la ligne d'en-tête du tableau

< tr>
< th>Nom< /th>
< th>Âge< /th>
< th>Pays< /th>
< /tr>

pour encadrer l'entète il faut ajouter a th séparément

td, th
{border: 1px solid black;}

caption

< table>
< caption> titre< /caption>
< tr>...< /tr> ...
< /table>

changer la position du titre

caption-side: top ou bottom

Un tableau structuré

Diviser un gros tableau

< thead>< /thead>la fusion de colonnes
< tbody>< /tbody>le corps (au centre)
< tfoot>< / tfoot>le pied du tableau (en bas)
il est conseillé d'écrire les balises dans l'ordre suivant : < thead> < tfoot> < tbody>
